Acid-Base Chapter Notes

1. Arrhenius, Lewis, and Bronsted-Lowry Theories

  • Arrhenius Definition:
    • Acid: Produces H+ ions in water.
    • Base: Produces OH- ions in water.
    • Example: HCl → H+ + Cl-, NaOH → Na+ + OH-.
  • Bronsted-Lowry Definition:
    • Acid: Proton donor.
    • Base: Proton acceptor.
    • Example: NH3 + H2O → NH4+ + OH-.
  • Lewis Definition:
    • Acid: Electron pair acceptor.
    • Base: Electron pair donor.
    • Example: BF3 + NH3 → F3B-NH3.

2. Will Salt be Acidic or Basic?

3. Difference Between Ka and pKa

  • Definitions:
    • Ka: Acid dissociation constant.
    • pKa = -log Ka: Inverse log scale for acidity strength.
  • Relationship:
    • Lower pKa → Stronger acid.
    • Example: Ka(HCl) > Ka(CH3COOH), so HCl is stronger.

4. Four Types of Oxides

  • Acidic Oxides: React with bases (e.g., CO2 + 2NaOH → Na2CO3 + H2O).
  • Basic Oxides: React with acids (e.g., Na2O + HCl → 2NaCl + H2O).
  • Amphoteric Oxides: React with acids and bases (e.g., Al2O3).
  • Neutral Oxides: Do not react with acids or bases (e.g., CO, N2O).

5. pKa and pKb Math

  • Key Formula:
    • pKa + pKb = 14 (for conjugate acid-base pairs).
  • Example:
    • If pKa = 4.76, then pKb = 14 - 4.76 = 9.24.

6. Ka and Kb Math

  • Formula:
    • Ka · Kb = Kw = 10-14 (at 25°C).
  • Example:
    • If Ka = 1.8 × 10-5, find Kb:
    • Kb = 10-14 / 1.8 × 10-5 = 5.56 × 10-10.
